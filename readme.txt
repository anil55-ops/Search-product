== Installation ==

= Installation from within WordPress =

1. Visit **Plugins > Add New**.
2. Search for **search-product**.
3. Install and activate the Plugin search-product plugin.

= Manual installation =

1. Upload the entire `search-product` folder to the `/wp-content/plugins/` directory.
2. Visit **Plugins**.
3. Activate the Plugin search-product.
